# Early-Detection-of-Millet-Disease
AI-powered millet disease detection system using MobileNetV2. Users upload leaf images to identify diseases and get symptoms, causes, and treatments. Includes scan history, analytics, and IoT support (ESP32 with soil, temperature, humidity sensors) for smart farming.”

The Millet Disease Detection System is an intelligent web-based application developed to identify diseases in millet crops using image processing and machine learning techniques. Millet crops are highly prone to various diseases that can significantly reduce yield and quality. Early detection is essential for preventing major losses, but many farmers still rely on manual observation, which is often inaccurate and time-consuming. This project aims to provide a fast, reliable, and user-friendly solution to this problem.
The system allows users to upload an image of a millet plant leaf through a simple interface. The uploaded image is processed using a trained deep learning model (MobileNetV2), which classifies whether the plant is healthy or affected by a specific disease. Once the prediction is made, the system displays detailed information including the disease name, symptoms, possible causes, and recommended treatment or recovery methods. This enables farmers to take timely and appropriate action.
In addition to disease detection, the system stores the history of all previous scans in a database. Users can view their past results and analyze disease patterns using basic charts and analytics. This feature helps in better understanding crop health over time and supports decision-making.
The project also incorporates basic hardware integration using sensors such as soil moisture, temperature, and humidity, connected through an ESP32 microcontroller. These sensors help monitor environmental conditions that influence plant health, making the system more effective and practical for real-world use.
Overall, the Millet Disease Detection System combines artificial intelligence and IoT concepts to support smart agriculture. It is designed with a clean and user-friendly interface, making it accessible even to users with minimal technical knowledge. This project helps improve crop productivity, reduce losses, and promote modern farming practices.
